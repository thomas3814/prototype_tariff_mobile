# v26 변경사항

## 반영 내용
- 모바일 `제품 수출국 % 기준 수입국 비교` 화면에서 국가명/bar/%를 감싸는 카드의 높이를 조금 더 줄였습니다.
- 병합된 복수 국가 라벨(예: `사우디/아랍에미리트`)은 더 작은 폰트로 표시되도록 조정했습니다.
- 기존의 bar 색상, 대륙 접기/펼치기, 라벨 터치 시 전체 이름 팝오버 동작은 유지합니다.

## 적용 범위
- 모바일 `수출국 1개 선택 / 수입국 전체 선택` 비교 화면
- PC 화면과 모바일 `수입국 기준 수출국 비교` 레이아웃은 이번 수정 범위에 포함하지 않았습니다.
