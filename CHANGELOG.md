# Changelog

이 프로젝트의 주요 변경사항은 이 문서에 기록합니다.

## [v16] - 2026-03-14

### Changed
- PC/모바일 전환을 위한 하단 화면 모드 스위치 추가
- 모바일 모드에서 `수입국 기준 수출국 비교` 그래프를 국가 10개 목록형으로 개선
- 모바일 목록에서는 `국가명 / bar / %`만 먼저 보이고, 국가 터치 시 상세 정보가 펼쳐지도록 변경
- 모바일 상세 영역에서만 `원산지 기준`, `HS 코드`, `협정관세 / 기본관세 / 적용관세`를 노출하도록 조정
- PC 모드의 기존 그래프 영역은 유지

### Notes
- 모바일 그래프 요약값은 국가별 표시관세 기준으로 표시
- 수출국 개수가 10개 미만인 제품은 실제 보유 국가 수만큼만 노출
