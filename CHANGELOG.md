# Changelog

이 프로젝트의 주요 변경사항은 이 문서에 기록합니다.

## [v26] - 2026-03-15

### Changed
- 모바일 `수출국 기준 수입국 비교`(`graph-by-import`)에서 국가명/bar/%를 감싸는 카드의 상하 여백과 내부 간격을 한 번 더 줄여 전체 세로 길이를 추가 압축
- 모바일 `수출국 기준 수입국 비교`에서 `사우디/아랍에미리트`처럼 병합된 복수 국가 라벨은 더 작은 폰트로 표시되도록 조정

### Notes
- 적용 범위는 모바일 `수출국 1개 선택 / 수입국 전체 선택` 비교 화면에 한정
- v25에서 추가된 복수 국가 라벨 터치 팝오버는 그대로 유지

## [v25] - 2026-03-15

### Changed
- 모바일 `수출국 기준 수입국 비교`, `수입국 기준 수출국 비교`에서 같은 조건으로 병합된 복수 국가 표기를 `국가1/국가2` 형식으로 정리
- 모바일에서 말줄임 처리된 복수 국가 라벨은 라벨 터치 시 전체 국가명을 팝오버로 표시하도록 보강
- 저장소 첫 방문자가 `Download ZIP`, `Fork`, `GitHub Pages` 흐름을 쉽게 이해할 수 있도록 루트 `README.md`, 앱 `README.md`, 초보자용 GitHub 가이드를 전면 정리

### Notes
- 모바일 복수 국가 전체 이름 표시는 bar 행 전체가 아니라 국가명 라벨 터치 시에만 노출
- Pages 배포 방식 안내는 현재 저장소의 GitHub Actions 배포 구조를 기준으로 정리


## [v23] - 2026-03-15

### Changed
- 모바일 `수입국 기준 수출국 비교`에도 대륙 단위 `접기 / 펼치기` 버튼과 상단 `대륙(n) 펼치기` 칩을 적용해 `수출국 기준 수입국 비교`와 동일한 조작 방식과 디자인으로 통일
- 모바일 `수입국 기준 수출국 비교`는 기존 상위 10개국 표시 정책을 유지하되, 표시 대상 10개국을 대륙별로 묶어 같은 방식의 접기/펼치기를 사용할 수 있도록 재구성
- 모바일 `수출국 기준 수입국 비교`에서 국가 bar 카드의 상하 여백과 내부 간격을 추가로 더 줄여 전체 스크롤 길이를 한 번 더 압축
- 모바일 `수출국 기준 수입국 비교`에서 같은 대륙 안의 국가들이 `HS / 원산지 기준 / 적용관세 / 협정관세 / 기본관세`까지 동일한 경우에는 `국가1 / 국가2` 형태의 단일 bar로 합쳐 표현하도록 개선

### Notes
- 대륙 버튼의 `(n)` 표시는 합쳐진 bar 개수가 아니라 실제 국가 수를 기준으로 유지
- 병합된 국가 bar를 눌렀을 때 상세 정보는 동일 조건을 대표하는 1세트만 펼쳐지며, PC 모드 그래프 구조는 유지

## [v22] - 2026-03-15

### Changed
- 모바일 `수입국 기준 수출국 비교`와 `수출국 기준 수입국 비교`에서 국가 bar 카드의 상하 여백과 바깥 컨테이너 여백을 더 촘촘하게 줄여 전체 세로 스크롤 길이를 축소
- 모바일 대륙 그룹 영역의 외곽 테두리와 내부 국가 bar 카드 간 간격을 좁혀 bar와 대륙 토글이 거의 붙어 보이도록 조정
- 모바일에서 우측 대륙 `접기` 버튼으로 접힌 그룹은 상단의 한 줄형 `대륙(n) 펼치기` 칩 버튼으로 모아 표시하고, 버튼은 내용 길이만큼만 감싸며 여러 개일 때 가로로 이어지다가 폭이 넘치면 다음 줄로 자동 줄바꿈되도록 개선
- 모바일 `대륙(n) 펼치기` 칩과 우측 `대륙(n) 접기` 버튼의 폰트를 추가로 축소해 버튼 좌우 폭과 세로 공간을 함께 압축
- 모바일 국가명 / bar / % 요약 행의 간격과 크기를 함께 미세 조정해 긴 목록에서 스크롤 밀도를 높임

### Notes
- 적용 범위는 모바일 `graph-by-export`, `graph-by-import` 그래프 영역이며, 대륙 그룹 칩 배치는 `graph-by-import`에서 사용
- PC 모드의 기존 그래프 구조와 대륙 접기/펼치기 동작은 유지

### Notes

## [v19] - 2026-03-14

### Changed

## [v18] - 2026-03-14

### Changed
- PC 모드에서 대륙을 접었을 때 `대륙명 / (국가수) / 펼치기`가 보이도록 접힘 카드 문구를 보강
- 모바일 모드에서 `수출국 기준 수입국 비교` case는 상위 10개국 제한을 제거하고 전체 국가를 모두 표시
- 모바일 `수출국 기준 수입국 비교`에 대륙 단위 `접기/펼치기` 기능을 추가해 긴 국가 목록을 정리해서 볼 수 있도록 개선
- 모바일 국가 요약 행은 기존과 동일하게 `국가명 / bar / %` 중심 구조를 유지하고, 터치 시 상세 정보만 펼쳐지도록 유지

### Notes
- 멕시코 예시 기준 수입국 비교는 `유럽(3), CIS(1), 북중미(3), 남미(5), 아시아(5), 중아(4)` 총 21개국이 모두 표시됨
- 모바일 대륙 접기 상태는 PC의 대륙 접기 상태와 동일한 키를 사용해 일관되게 동작
