## v15 - 디자인 보정 최종 완료

엑셀 업로드 기능 설계 프로젝트의 **v15**는 현재 기준 최신 안정 버전입니다.  
이번 버전에서는 디자인 보정이 최종 완료되었으며, 다른 사용자도 저장소를 쉽게 가져갈 수 있도록 README와 시작 가이드를 함께 정리했습니다.

### 주요 내용

- v15 기준 디자인 보정 최종 반영
- 저장소 첫 진입용 README 구조 개선
- Fork / Clone / ZIP 다운로드 가이드 추가
- 공유 및 재사용 관점의 문서 보강

### 추천 사용 방식

- 빠른 검토: `Code` → `Download ZIP`
- 로컬 개발: `git clone`
- 별도 운영/개선: `Fork`

### 사용 대상

- 내부 검토용 샘플이 필요한 팀
- 업로드 UX / 화면 구조를 참고하려는 개발자
- 커스터마이징 가능한 시작점을 찾는 프로젝트 담당자

### 참고 문서

- `README.md`
- `docs/GETTING_STARTED.md`
- `CONTRIBUTING.md`
- `CHANGELOG.md`

### 운영자 메모

Release 발행 시 아래 항목을 함께 정리하면 전달력이 좋아집니다.

- 샘플 엑셀 또는 템플릿 위치
