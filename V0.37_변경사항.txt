V0.37 변경사항

1) 제품 수입국 % 기준 수출국 비교 (모바일)
- 수입국 기준 수출국 비교 모바일 화면의 폰트 크기를 전반적으로 한 단계 축소
- 모바일 비교 헤더 제목, 상단 요약 범례, 카드 요약 박스/국가명/상세 영역 폰트 축소
- 1개 국가만 남는 마지막 대륙(CIS 등)은 가운데 정렬 대신 좌측 슬롯에 배치되도록 변경
- 이에 따라 CIS의 협정/기본 박스 위치가 유럽, 중아와 같은 세로 기준선으로 정렬되도록 조정

2) 공통 적용 범위
- 멕시코 사례뿐 아니라 다른 수입국 선택 시에도 동일한 모바일 정렬 로직 적용
- 기존 카드 펼침/접기, 색상 규칙, 채택 표시 로직은 유지

3) 검증
- npm ci 완료
- npm run lint 완료
- npm run build 완료
