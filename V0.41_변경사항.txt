[V0.41 변경사항]

1. 제품 수출국 % 기준 수입국 비교 (PC)
- 수출국 멕시코 선택 시와 같이 PC 비교 셀에 `n.n%`, `n.n% or n.n%` 형태의 긴 관세 값이 들어올 때 `...` 대신 font size가 더 적극적으로 축소되도록 조정
- 선택된 관세 셀은 채택 배지 영역까지 고려하여 글자 폭을 다시 계산하도록 보정
- compact 비교 셀의 내부 여백과 배지 크기를 조정하여 긴 문자열이 더 안정적으로 한 줄 표시되도록 개선

2. 제품 수입국 % 기준 수출국 비교 (PC)
- 수입국 멕시코 선택 시와 같이 PC 비교 셀에 `n.n%`, `n.n% or n.n%` 형태의 긴 관세 값이 들어올 때 동일하게 font size 축소 적용
- 다른 국가/대륙의 긴 관세 문자열에도 동일한 자동 축소 로직이 공통 적용되도록 정리

3. 공통 텍스트 fitting 로직 개선
- 관세 값 길이에 따라 비율 기반으로 font size를 먼저 계산한 뒤, 필요한 경우 더 미세하게 축소하도록 개선
- PC compact 셀에서는 말줄임 대신 clip 모드로 처리하여 `...`가 노출되지 않도록 조정

4. 버전 반영 및 배포 산출물 포함
- 하단 버전 표기 `v 0.41` 반영
- tariff-pages-app/dist 빌드 완료
- 루트 docs 폴더에 dist 결과 복사 완료

5. 검증
- npm run lint 완료
- npm run build 완료
