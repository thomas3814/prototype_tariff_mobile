V0.36 변경사항

1) 제품 수입국 % 기준 수출국 비교 (모바일)
- 1개 국가만 있는 대륙 섹션을 2개씩 묶어 좌/우 슬롯으로 배치하도록 정렬 규칙 추가
- 좌측 카드와 우측 카드를 각각 바깥쪽으로 맞춰, 아래 다국가 대륙(예: 아시아) 카드의 협정/기본 위치와 세로 기준선이 더 잘 맞도록 조정
- 1개 대륙만 남는 경우는 가운데 정렬 유지

2) 공통 적용 범위
- 수입국 멕시코 사례뿐 아니라, 다른 수입국 선택 시에도 동일한 모바일 정렬 로직 적용
- 기존 카드 크기, 토글 동작, 색상 및 상세 펼침 동작은 유지

3) 검증
- npm ci 완료
- npm run lint 완료
- npm run build 완료
